The brief's daily timing should set the start time for Day 2 onward

…and the arrival day starts when the traveller lands — not at the wake time. Try the timing buttons.

The rule you asked for

  • Day 1 (arrival day) starts at the arrival time — they've just landed, so no morning wake anchor.
  • Day 2, Day 3, … (every day after arrival) start at the daily-timing wake time shown in the brief.
  • So if the brief says 標準・08:30 → Day 2+ start at 08:30. If it says 遅め・09:30 → Day 2+ start at 09:30. The brief and the days always agree.
